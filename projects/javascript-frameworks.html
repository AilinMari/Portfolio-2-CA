Overview

This project is a modern e-commerce application built with React, TypeScript, and Vite. It is designed around reusable components and a clear shopping journey from product discovery to checkout.

The implementation emphasizes maintainability through structured state management, predictable routing, and typed development workflows.

  • Product listing and product detail views.
  • Cart functionality with add/remove item flows.
  • Checkout experience with success confirmation.
  • Contact form with validation and user feedback messaging.
  • Shared state management for cart and UI notifications.

Tech stack: React, TypeScript, Vite, and REST API integration.
