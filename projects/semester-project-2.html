Overview

Golden Bid is a full-featured online auction platform developed for Semester Project 2. The application is designed to support the full auction journey, from creating listings to placing bids and managing profile activity in real time.

The experience is built around clear navigation, quick item discovery, and real-time interaction feedback across desktop and mobile devices.

  • User authentication and personalized profiles.
  • Create, edit, and manage auction listings.
  • Live bidding with countdown timers.
  • Search and filtering for faster item discovery.
  • Bid history and user activity tracking.
  • Responsive, mobile-first layout.

Tech stack: JavaScript, Tailwind CSS, Node.js, and REST API integration.
