Overview

This project is a responsive social media prototype built with HTML5 and Tailwind CSS. It explores how a utility-first framework can speed up development while keeping the interface consistent across pages.

The final result focuses on visual hierarchy, spacing consistency, and mobile-first behavior, with layouts that scale smoothly across breakpoints.

  • Authentication page with form-focused layout patterns.
  • Dynamic feed with search and sorting functionality.
  • User profile page with reusable UI sections.
  • Responsive behavior handled through Tailwind breakpoints.
  • Component-like utility patterns for faster styling decisions.

Tech stack: HTML5, Tailwind CSS, PostCSS, and NPM tooling.
